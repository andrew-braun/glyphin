# N'Ko Reading Lesson Sequence

This staged outline is a review-ready starting point, not final learner-facing
lesson copy. It assumes N'Ko literacy for a Manding-language context and requires
expert review before implementation.

## Course Shape

- First-session reading win: read `ߒߞߏ` as the script-name anchor.
- Approximate lesson count: 12 core lessons plus reviewer-selected domain text.
- Structural family: RTL alphabet with vowel letters, combining tone and nasal
  marks, digits, and punctuation.
- Required app support beyond the current Thai runtime: RTL layout, bidi
  isolation, combining mark display, font coverage, tone-mark metadata, and
  language-scope notes.

## Coverage Notes

- Target-domain samples: community literacy materials, dictionary headwords,
  digital text, cultural identity text, and short public labels.
- Known-unit coverage goal: establish RTL order, script identity, core letters,
  and the first mark policy before wider vocabulary.
- Known-anchor coverage goal: approve `ߒߞߏ` and at least two short anchors before
  runtime implementation.
- Deferred high-load material: full tone-mark pedagogy, advanced orthography,
  language-variety comparison, production spelling, copied external text, and
  non-Manding generalization.
