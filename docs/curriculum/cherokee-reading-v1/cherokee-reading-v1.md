# Cherokee Reading Curriculum Notes

This durable course note records the v1 boundary and bootstrap decisions for a
Cherokee syllabary reading course.

## Course Boundary

- Language/script: Cherokee in the Cherokee syllabary, `chr-Cher`, ISO 15924
  `Cher`.
- Target learner: heritage learner, community ally, museum visitor, or script
  enthusiast who wants to recognize respectful short Cherokee text.
- Target domains: community language signage, cultural-site labels, dictionary
  headwords, language-learning materials, and short public labels.
- In scope for v1: modern printed syllabary, a small high-payoff set of
  syllabograms, identity and greeting anchors, and reviewer-approved app-authored
  examples.
- Out of scope for v1: full language instruction, dialect comparison, complete
  chart memorization, handwriting, production spelling, and unreviewed copied
  examples.

## Sequencing Rationale

- Frequency sources: Unicode and CLDR metadata for inventory, community language
  resources for pedagogy discovery, Wiktionary for cautious spelling checks, and
  app-authored examples for shipped content.
- First-session decoding target: `ᎣᏏᏲ`, `ᏩᏙ`, and `ᏣᎳᎩ` because they are
  compact and culturally central but must still be expert-reviewed.
- Stage 1 goal: teach that Cherokee signs are syllables and give learners a few
  meaningful reading wins.
- Stage 2 goal: add sign-family sequencing and a small set of community-label
  candidates.
- Stage 3 goal: handle case policy, font testing, and reviewer-selected domain
  text before implementation.

## Script Notes

- Cherokee has a large syllabary inventory and modern uppercase/lowercase case
  behavior; v1 should not assume lowercase belongs in first lessons.
- Many signs have Latin lookalikes. Drills need to teach Cherokee identity
  directly instead of leaning on Latin mnemonics.
- Community review is not optional because language-revitalization context affects
  word choice, glosses, and tone.

## Validation Notes

- Segmentation review: verify syllabogram boundaries, case variants, and any
  punctuation in sample text.
- Pronunciation review: confirm each anchor and syllable value with Cherokee
  language experts.
- Cultural review: validate that greeting, gratitude, identity, and public-label
  examples are appropriate for learner display.
- License review: keep external resources discovery or scoring only until terms
  are approved.
