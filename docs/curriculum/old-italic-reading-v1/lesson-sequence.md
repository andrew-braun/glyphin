# Old Italic Reading Lesson Sequence

This staged outline targets right-to-left Old Italic recognition with an
Etruscan-leaning inscription boundary. It needs specialist review before any
learner-facing publication.

## Course Shape

- First-session reading win: read `𐌌𐌉` as a short reviewed word while noticing
  right-to-left orientation.
- Approximate lesson count: 12 lessons plus optional name and alphabet-table
  review.
- Structural family: alphabetic historic script family with RTL orientation,
  inscriptional spacing, multiple regional alphabets, and transliteration needs.
- Required app support beyond the current Thai runtime: course-level RTL display,
  transliteration fields, inscription provenance, orientation drills, and Old
  Italic font fallback.

## Coverage Notes

- Target-domain samples: Etruscan funerary inscriptions, museum object labels,
  personal names, alphabet tables, and transliteration workbenches.
- Known-unit coverage goal: by Lesson 8, learners can read right-to-left strings
  with core vowels, common consonants, kinship, funerary, and name anchors.
- Known-anchor coverage goal: learners recognize a small inscription-domain set
  without assuming all Old Italic alphabets behave identically.
- Deferred high-load material: non-Etruscan Old Italic alphabets, boustrophedon
  examples, exact inscriptions, punctuation, dialect variation, and grammar.
