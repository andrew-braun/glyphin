# Runic Reading Lesson Sequence

This staged outline is a review-ready starting point, not final learner-facing
lesson copy. It uses a cautious Elder Futhark first pass and labels every value
as a transliteration convention rather than a single living pronunciation.

## Course Shape

- First-session reading win: read `ᚠᚢᚦᚨᚱᚲ` as the start of the futhark and identify
  a few high-visibility rune shapes.
- Approximate lesson count: 12 core lessons plus optional period-specific
  expansions.
- Structural family: alphabetic script with period and language variants,
  transliteration conventions, separators, and inscriptional direction notes.
- Required app support beyond the current Thai runtime: transliteration display,
  rune-variant metadata, historic-period labels, separator handling, and
  reviewer-gated sensitivity notes.

## Coverage Notes

- Target-domain samples: museum labels, runestone catalogs, beginner futhark
  charts, replica artifact labels, and scholarly transliteration examples.
- Known-grapheme or known-unit coverage goal: by Lesson 7, learners can read the
  first futhark row plus high-payoff runes used in common teaching names.
- Known-word or known-anchor coverage goal: learners can decode futhark and a
  controlled set of rune-name anchors without assuming one historical language.
- Deferred high-load material: Younger Futhark and Anglo-Saxon variants,
  manuscript runes, bind-runes, exact inscription readings, separators, local
  dating, and modern ideological misuse handling beyond required review notes.
